# zomato-powerbi-analysis
Interactive Power BI dashboard analyzing Zomato restaurant data to uncover customer preferences, ratings, pricing trends, and business insights.
The Zomato Restaurant Analysis Dashboard is an interactive Power BI project developed to analyze restaurant performance, customer preferences, pricing trends, and ratings using the Zomato dataset.
The dashboard transforms raw data into meaningful business insights through interactive visualizations and key performance indicators (KPIs).

Key Features:
Interactive Power BI dashboard with dynamic filters and slicers
Analysis of restaurant ratings, votes, and pricing
City-wise and cuisine-wise performance analysis
Online delivery and table booking analysis
